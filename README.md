# AI vs Employee Cost Calculator

This lead magnet tool helps businesses visualize the cost savings of replacing repetitive employee tasks with AI agents.

## How It Works

The calculator compares:
- Traditional employee costs (salary, benefits, overhead)
- AI agent costs (software subscriptions, minimal maintenance)

## Key Features

1. **Simple Inputs:**
   - Monthly salary
   - Hours worked per month
   - Tasks handled per month

2. **Instant Calculations:**
   - Cost per task (employee vs AI)
   - Monthly and annual savings potential
   - Break-even analysis

3. **Viral Elements:**
   - Screenshot-worthy results
   - Clean, professional design
   - Mobile-responsive layout
   - Shareable outcomes

## Business Example

We've included a compelling example showing how replacing 8 employees with AI agents can save:
- $40,000/month in salaries
- Only $800/month for AI agents
- Resulting in $39,200/month ($468,000/year) in savings

## How to Use

1. Simply open `index.html` in any browser
2. Adjust the input values to match your scenario
3. Click "Calculate Savings" to see results
4. Share screenshots on social media or in presentations

## Customization

To customize for your specific market or pricing:
1. Edit the HTML file
2. Modify the example values in the "8-Agent Setup" section
3. Update the Calendly link to your own booking page
4. Adjust colors in the CSS variables if desired
